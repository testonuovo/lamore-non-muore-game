# ✈️ L’Amore Non Muore – Flying Plane Game 🎵

## 💖 Descrizione
Questo è un gioco HTML sviluppato da **Singer Orest**, dove controlli un aereo che deve evitare ostacoli.  
Il sottofondo musicale è la mia canzone originale **“L’Amore Non Muore”**.  

Il gioco aumenta di difficoltà man mano che si avanza nei livelli e mostra punteggio e livello attuale.

---

## 🎮 Istruzioni di gioco
- Premi **Space** per far volare l’aereo  
- Evita gli ostacoli rossi  
- Raggiungi il punteggio più alto possibile  
- La difficoltà aumenta progressivamente man mano che giochi
